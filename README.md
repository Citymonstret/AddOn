# AddOn
Simple java AddOn system
```xml
<repositories>
	<repository>
	    <id>jitpack.io</id>
	    <url>https://jitpack.io</url>
	</repository>
</repositories>

<dependency>
    <groupId>com.github.Sauilitired</groupId>
    <artifactId>AddOn</artifactId>
    <version>0.0.2</version>
</dependency>
```
